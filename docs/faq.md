# FAQ

## What is Mangarr?

Mangarr is a manga library manager that helps you organize and track your
manga collection.

## Where can I get help?

Join the [Mangarr Discord](https://mangarr.github.io/discord) or open an issue
on [GitHub](https://github.com/mangarr).

## How do I contribute to this wiki?

Every page has an edit icon in the top-right that links to the source Markdown
file on GitHub. Open a pull request with your changes and they'll be reviewed.
