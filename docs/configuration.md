# Configuration

Reference for configuring Mangarr.

## Settings overview

| Setting | Description | Default |
| --- | --- | --- |
| `port` | Port the web UI listens on | `8080` |
| `library_path` | Where your manga library is stored | `/data` |
| `log_level` | Verbosity of logs | `info` |

!!! warning
    These values are placeholders. Update this page as configuration options
    are finalized.

## Environment variables

```bash
MANGARR_PORT=8080
MANGARR_LIBRARY_PATH=/data
MANGARR_LOG_LEVEL=info
```
